Regla clave

Si multiplicas o divides por un número negativo, el signo cambia.

Ejemplo básico

2x + 3 > 7

x > 2

Ejemplo intermedio

-3x + 6 ≤ 0

-3x ≤ -6

x ≥ 2

Ejemplo avanzado

(x - 2)(x + 1) > 0

Solución: x < -1 o x > 2

Grado relativo y general

Grado relativo: exponente de una variable.

Grado general: suma de exponentes.
